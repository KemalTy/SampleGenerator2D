------------------
What is this?

This is a simple Matlab tool that will let you create points or point-label pairs in 2D with a GUI.

------------------
What is it good for?

You can generate 2D points, with their labels if desired, using a simple GIU. The data generated with this tool can be used to test and debug machine learning codes.

------------------
How to use it?

To generate unlabeled data, run "unlabeledDataCreator.m". To generate labeled data, run "labeledDataCreator.m". When you run, GUI will pop up and walk you through the process.

------------------
Are there any restrictions?

This tool works only with Matlab version below 2014b.
